🚀 Surya Karthik — Personal Portfolio Website
A clean, responsive personal portfolio website built with HTML, CSS, and JavaScript. Designed with a dark theme and deployed on Vercel.
🔗 Live Site: your-site.vercel.app

👨‍💻 About
This is my personal portfolio website showcasing my skills, projects, experience, and certifications as an Aspiring Data Scientist.

✨ Features

Dark theme UI with blue accent highlights
Responsive design — works on mobile, tablet, and desktop
Smooth scroll navigation
Animated skill bars
Project/Work showcase section
Certificate carousel with manual swipe
Contact form (powered by Formspree)
Resume viewer (opens in new tab)


🛠️ Built With

HTML5
CSS3
JavaScript (Vanilla)
Boxicons — for icons
Formspree — for contact form emails
Google Fonts — Poppins
